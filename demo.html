🎉 My Blog - 个人博客系统

一个功能完整的现代化博客平台，支持 Markdown 编辑、图片上传、搜索和分页功能

查看源码 部署指南

✨ 功能亮点

支持所有现代化博客功能，让写作变得简单有趣

📝
Markdown 编辑器

支持标题、加粗、斜体、代码块、列表等所有 Markdown 语法，实时预览功能让写作更直观。

🖼️
图片上传

支持拖拽上传文章配图和个人头像，轻松管理博客中的视觉内容。

🔍
搜索功能

实时搜索文章标题和内容，快速找到你想要的内容。

📄
分页显示

文章列表自动分页，当文章增多时也能保持良好的浏览体验。

🌙
深色模式

支持明暗主题切换，保护眼睛，适应不同使用环境。

💬
评论系统

内置评论功能，让访客可以与你交流互动。

🚀 快速开始

1

克隆仓库

下载或克隆项目到本地：
git clone https://github.com/Chuan32/my-blog.git

2

启动本地服务器

进入项目目录，运行：
cd my-blog
python -m http.server 8000

3

访问博客

打开浏览器访问：
http://localhost:8000

4

开始写作

访问 admin.html 登录并发布你的第一篇文章！

⚠️ 注意事项

  • 图片上传功能需要后端支持，目前是纯前端演示
  • 所有数据存储在浏览器本地，清除数据会丢失
  • 建议部署到 GitHub Pages 或 Netlify 以获得完整体验
